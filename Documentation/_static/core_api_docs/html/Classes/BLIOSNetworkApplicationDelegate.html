Overview

The delegate for a networked app. BLIOSNetworkApplicationDelegate is the application delegate responsible for creating and configuring the app’s root view controller and the navigation controller that will contain it. It supports additional networking functionality by providing methods for detecting wifi/gsm connectivity and events for responding to changes in that connectivity.
