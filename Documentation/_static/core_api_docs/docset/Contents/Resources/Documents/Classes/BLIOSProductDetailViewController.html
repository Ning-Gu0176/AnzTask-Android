Overview

The view controller for displaying the details of a product.

Bug: The *go back to your basket" button needs to have color to make it more visible and it’s method needs to be implemented.

Bug: The image needs to be smaller on an iPhone/iPod touch to allow more room for a long product description.

Bug: The price label is not updated when a product price is changed.
