Google Volley
=============

Refer Google Volley’s website for more information:
http://developer.android.com/training/volley/index.html

To clone Volley source code and generate Volley jar file, do following command lines in terminal:

 - git clone https://android.googlesource.com/platform/frameworks/volley
 - cd volley
 - android update project -p .
 - ant jar
 - find volley.jar in /bin folder
