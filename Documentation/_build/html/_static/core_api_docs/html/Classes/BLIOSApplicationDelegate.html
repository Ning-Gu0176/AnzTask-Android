Inherits from BLObject : NSObject
Conforms to UIApplicationDelegate
Declared in BLIOSApplicationDelegate.h

Overview

The BLIOSApplicationDelegate is the application delegate responsible for creating and configuring the app’s root view controller and the navigation controller that will contain it.
