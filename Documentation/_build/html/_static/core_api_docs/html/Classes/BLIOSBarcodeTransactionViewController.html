Overview

The view controller for adding a product to a transaction by scanning a barcode.

Bug: The view referenced by onBarcodeScanned:fromOrigin:withView needs to have it’s display updated to feed back the product data retrieved from the shopping agent.

Bug: The offers encapsulated in a product need to be taken into account and reflected in the total when proceeding to checkout.
