onIsMerchantConfigured:forRetailer:

Initiated to determine whether the merchant data is configured on the device.

- (bool)onIsMerchantConfigured:(bool)aHasPaymentDevice forRetailer:(BLRetailer *)aRetailer

Parameters

aHasPaymentDevice.

Indicates whether the mobile device is attached to a payment device. If true then the mobile device is attached to a payment device. Otherwise, the mobile device is not attached to a payment device.

Return Value

True if the merchant is configured. Otherwise false.
