onProducts:retrievedForBasket:onDate:withNextCount:fromSource:usingSearch:shouldOverwrite:

Initiated when products have been retrieved from the shopping agent.

- (void)onProducts:(NSArray *)aProducts retrievedForBasket:(BLBasket *)aBasket onDate:(NSString *)aDate withNextCount:(NSInteger)aNextCount fromSource:(BLSearchOrigin)aOrigin usingSearch:(NSString *)aSearchString shouldOverwrite:(bool)aOverwrite

Parameters

aProducts

The products that were retrieved from the shopping agent.

aBasket

The basket associated with the initial request.

aDate

The date that the product inventory was last updated.

aNextCount

The number of remaining products to be retrieved as a result of the originating request.

aOrigin

The source type that will determine how the products are processed.

aSearchString

The product barcode or product name that was used to initiate the search.

aOverwrite

Determines whether the results of a previous search should be overwritten. If true then the previous results are overwritten. Otherwise, the retrieved products are appended to the previous results.
